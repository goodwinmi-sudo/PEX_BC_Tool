# UX Design Guidelines: Aesthetic & Usability

As the UX Designer, my mission is to ensure the PEX BC Tool is intuitive, visually stunning, and provides a premium experience for Googlers.

## Design Philosophy

"Premium, Minimalist, and Data-Driven."

## UI Requirements

1. **Consistency**: Use the established Tailwind color palette (Gray-900 for background, Blue-400 for accents).
2. **Micro-interactions**: Use transitions and hover effects to make the tool feel "alive."
3. **Accessibility**: Maintain high contrast and clear typography (Sans-serif).
4. **Hierarchy**: Use clear font sizes and spacing to emphasize critical information.

## Governance View (/board)

The `/board` view must present a professional, administrative dashboard layout, distinguishing it from the "simulation" views.

> [!TIP]
> Use glassmorphism and subtle gradients to enhance the "premium" feel.
